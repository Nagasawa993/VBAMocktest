【9章】　エラー対応

　

Q9_1)　次のコードは、もしエラーが発生したら、ラベルError1に処理をジャンプさせるマクロである。 [空欄]に入る適切な語句を入力しなさい。 

        
      Sub Macro1()
      　On Error [空欄] Error1
      　Sheets("Sheet1").Name = Range("A1")
      　Exit Sub
      Error1:
      　MsgBox "エラーが発生しました"
      End Sub
　答え：　

　

Q9_2)　次のコードは、エラーが発生してもエラーを無視して処理を続け、メッセージを表示するマクロである。 [空欄]に入る適切な語句を入力しなさい。 

        
          Sub Macro1()
          　On Error [空欄]
          　Sheets("Sheet1").Name = Range("A1")
          　If Err.Number > 0 Then
          　　MsgBox "エラーが発生しました"
          　End If
          End Sub
